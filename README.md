# ENDE-UD04-A2

## Descripción

Este repositorio contiene la actividad **UD04-A2** de la asignatura **Entornos de Desarrollo (ENDE)**.
El objetivo de esta práctica es aprender a trabajar con repositorios remotos utilizando **Git** y plataformas de alojamiento de código como **GitHub**, **GitLab** o **Bitbucket**, así como familiarizarse con la gestión básica de proyectos y documentación.

* **Git** – Sistema de control de versiones.

* **Markdown** – Lenguaje de marcado para documentación.
